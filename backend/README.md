## Backend (JHipster) — backend/

Este directorio contendrá la aplicación servidor generada con 
JHipster.

Resumen
-------
Aplicación monolito generada con JHipster (Maven + JWT). Se usa Redis para cache/sesiones y PostgreSQL en producción; H2 (disk) en desarrollo.

Qué incluye
-----------
- backend/.yo-rc.json — configuración para reproducir la generación de la app con JHipster.
- Al generar la app con `jhipster` se creó la estructura estándar (src/, pom.xml, etc.).
